## Color

| Role | Value | Usage |
|------|-------|-------|
| Dominant (60%) | slate-950 | Background, surfaces |
| Secondary (30%) | slate-900/60 | Cards, wizard panels, list items |
| Accent (10%) | teal-400 / teal-500 | Primary CTAs, active wizard steps |
| Destructive | red-500 | Validation errors, remove teammate buttons |
